This easy home loan calculator will help you calculate monthly mortgage payments. The mortgage calculator is perfect for both fixed-rate or adjustable-rate home loans.

[https://www.calculator.io/home-loan-calculator/](https://www.calculator.io/home-loan-calculator/)

== Usage ==

Add the Home Loan Calculator shortcode to your page, post or sidebar:

`[ci_home_loan_calculator]`

Add the following code to your template where you would like the Home Loan Calculator to appear:

`<?php display_ci_home_loan_calculator(); ?>`

== Installation ==

1. Upload the Home Loan Calculator /ci_home_loan_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Home Loan Calculator plugin through the "Plugins" menu in WordPress.

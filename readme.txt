Easily estimate your monthly mortgage payments with our free Home Loan Calculator. View amortization schedules for fixed and adjustable-rate loans instantly.

[https://www.calculator.io/home-loan-calculator/](https://www.calculator.io/home-loan-calculator/)

== Usage ==

Add the Home Loan Calculator shortcode to your page, post or sidebar:

`[calcio_home_loan_calculator]`

Add the following code to your template where you would like the Home Loan Calculator to appear:

`<?php display_calcio_home_loan_calculator(); ?>`

== Installation ==

1. Upload the Home Loan Calculator /calcio_home_loan_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Home Loan Calculator plugin through the "Plugins" menu in WordPress.
